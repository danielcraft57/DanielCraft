# Prompts pour generer l'image de chaque projet (loupix57)

Utilise ces prompts dans un outil de generation d'images (DALL-E, Midjourney, Stable Diffusion, etc.) pour creer une illustration par projet. Chaque prompt est redige pour coller au **style et aux couleurs du site DanielCraft**.

---

## Charte visuelle du site (a respecter dans tous les prompts)

- **Couleur principale** : rouge vif `#dc2626` (boutons, accents, elements cles). Variantes : rouge plus fonce `#b91c1c`, rouge plus clair `#ef4444`.
- **Fonds** : blanc `#ffffff`, gris tres clair `#f9fafb`, `#f3f4f6`, ou degrade doux type "hero" (gris clair vers blanc). Pour les scenes tech/data : gris fonce `#374151`, `#1f2937`, ou noir doux `#111827`.
- **Neutres** : gris `#6b7280`, `#4b5563` pour textes secondaires et lignes.
- **Style** : moderne, epure, professionnel. Coins arrondis (8 a 16px), ombres legeres, pas de surcharge. Typographie type Inter (sans-serif propre et lisible). Ambiance "portfolio developpeur freelance" : serieux mais abordable, tech mais pas froid.
- **Composition** : equilibree, aerée. Laisser de la place pour un titre ou un sous-texte (en bas ou en coin). Pas de visages reconnaissables. Eviter : fouillis, trop de couleurs, look stock-photo, 3D kitsch.
- **Format** : 1200x630 px (ratio 1.91:1) pour partage social ; 800x450 pour cartes portfolio. Haute lisibilite en miniature.

## Utilisation

1. Copier le prompt du projet souhaite (tout le bloc entre les backticks).
2. Coller dans ton outil (DALL-E, Midjourney, Stable Diffusion, etc.).
3. Precise le ratio : **1200x630** ou **1.91:1** pour partage social ; **800x450** pour cartes portfolio.
4. Sauvegarder l'image dans `assets/images/projets/NOM_DU_PROJET.jpg` (ou .png).
5. Dans `portfolio.js`, ajouter `imageUrl: 'assets/images/projets/NOM.jpg'` au projet concerne. Sur la page /projets, lier l'image de la meme maniere si le rendu des cartes utilise une image.

## Fichiers concernes

- **Portfolio (accueil)** : `assets/js/portfolio.js` – champs optionnels `imageUrl` par projet.
- **Page Projets** : `assets/js/github-projects.js` – donnees loupix57 avec `account: 'loupix57'`. Si tu ajoutes un champ `imageUrl` aux entrees, adapte le rendu dans `projets-page.js`.
- **Clone des repos** : `scripts/clone-loupix57-repos.ps1` pour telecharger tous les depots loupix57 en local et les etudier.

# Prompt pour générer l'image Open Graph par IA

## Prompt pour Midjourney / DALL-E / Stable Diffusion

```
Create a professional, modern social media image for a freelance web developer portfolio. 

Technical specifications:
- Dimensions: 1200x630 pixels (1.91:1 aspect ratio)
- Format: High-quality JPEG
- Style: Clean, modern, tech-focused

Visual elements:
- Background: Dark gradient (deep blue to black) or subtle geometric pattern
- Central element: Large, bold text "DanielCraft" in white or bright accent color
- Subtitle: "Développeur Full-Stack TypeScript" in smaller text below
- Tech icons: Subtle, minimalist icons representing TypeScript, React, Node.js (small, not overwhelming)
- Accent: A single bright accent color (red #dc2626 or electric blue) for highlights
- Typography: Modern sans-serif font (Inter or similar), bold and readable
- Layout: Text centered, balanced composition, professional spacing

Mood and tone:
- Professional but approachable
- Tech-forward but not intimidating
- Clean and minimalist
- High contrast for readability when shared on social media

Avoid:
- Cluttered design
- Too many colors
- Low contrast text
- Overly complex graphics
- Stock photo feel

The image should look professional when shared on LinkedIn, Facebook, Twitter, and other social platforms.
```

## Prompt pour ChatGPT / Claude (avec génération d'image)

```
Génère une image Open Graph (1200x630px) pour mon site portfolio de développeur freelance.

Contenu :
- Titre principal : "DanielCraft" (grand, en gras, blanc ou couleur accent)
- Sous-titre : "Développeur Full-Stack TypeScript" (plus petit, gris clair)
- Icônes technos discrètes : TypeScript, React, Node.js (petites, en transparence)
- Couleur principale : Rouge #dc2626 ou bleu électrique
- Fond : Dégradé sombre (bleu foncé vers noir) ou motif géométrique subtil

Style :
- Moderne, épuré, professionnel
- Haute lisibilité pour le partage social
- Contraste élevé
- Typographie : Inter ou police moderne sans-serif
- Composition centrée et équilibrée

Format : JPEG haute qualité, 1200x630px, ratio 1.91:1
```

## Prompt pour Canva (si tu préfères créer manuellement)

1. Template : "Facebook Post" ou "LinkedIn Post"
2. Dimensions : 1200x630px
3. Éléments à ajouter :
   - Fond : Dégradé sombre ou couleur unie
   - Texte principal : "DanielCraft" (police Inter, gras, blanc)
   - Texte secondaire : "Développeur Full-Stack TypeScript" (gris clair)
   - Icônes : TypeScript, React, Node.js (petites, discrètes)
   - Couleur accent : #dc2626 (rouge)
4. Export : JPEG, qualité maximale

## Outils recommandés

- **Canva** : Template "Facebook Post", facile et rapide
- **Figma** : Plus de contrôle, gratuit
- **Midjourney** : Si tu veux quelque chose d'unique
- **DALL-E 3** : Via ChatGPT Plus, bon résultat
- **Stable Diffusion** : Si tu as une carte graphique

## Vérification

Une fois l'image créée :
1. Vérifie les dimensions : 1200x630px exactement
2. Poids : < 1MB (optimise si nécessaire avec TinyPNG)
3. Nom : `og-image.jpg`
4. Emplacement : `assets/images/og-image.jpg`
5. Test : Partage une URL sur Facebook pour voir le rendu

## Test de l'image

Une fois l'image en place, teste avec :
- [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/)
- [Twitter Card Validator](https://cards-dev.twitter.com/validator)
- [LinkedIn Post Inspector](https://www.linkedin.com/post-inspector/)

# Prompt pour générer les images Open Graph par IA

## Prompt pour Midjourney / DALL-E / Stable Diffusion

```
Create a professional, modern social media image for a freelance web developer portfolio called "DanielCraft", visually consistent with the homepage of danielcraft.fr.

Technical specifications:
- Dimensions: 1200x630 pixels (1.91:1 aspect ratio)
- Format: High-quality JPEG
- Style: Clean, modern, business-focused, with strong red accent

Visual elements:
- Background: Very light gray or off-white, clean and minimal, similar to a modern landing page
- Main area: A wide, soft card or band with a strong red accent color (#dc2626), slightly rounded corners and subtle shadow, echoing a hero section
- Central text: Large, bold word "DanielCraft" in a modern sans-serif font (similar to Inter), in very dark gray or almost black
- Subtitle: "Développeur Full-Stack TypeScript, Python & PHP" in smaller text below, in medium or dark gray
- Supporting elements: A few subtle badges or pill-shaped tags suggesting technologies (TypeScript, React, Node.js, Python, PHP) or benefits (Sites vitrines, SEO, IA & Data), using small icons or simple pictograms, aligned horizontally and kept very minimal
- Accent colors: Primary red #dc2626 for key elements (band, underline, small shapes, CTA-style block), with occasional neutral grays; no heavy blues
- Layout: Centered and balanced composition, generous white space, feeling like a clean hero section screenshot without real UI components

Mood and tone:
- Professional, trustworthy, and conversion-oriented
- Approachable and clear, focused on business value rather than pure tech
- Clean and minimalist, high contrast for readability when shared on social media

Avoid:
- Dark, fully black backgrounds
- Too many bright colors
- Low-contrast text
- Overly complex graphics or fake dashboards
- Stock photos or realistic people

The image should look like a modern landing-page hero visual, matching the identity of danielcraft.fr, and remain professional on LinkedIn, Facebook, Twitter, and other social platforms.
```

## Prompt pour ChatGPT / Claude (avec génération d'image)

```
Génère une image Open Graph (1200x630px) pour mon site portfolio de développeur freelance, visuellement cohérente avec la page d'accueil de danielcraft.fr.

Contenu :
- Titre principal : "DanielCraft" (grand, en gras, en très foncé, style Inter ou équivalent)
- Sous-titre : "Développeur Full-Stack TypeScript, Python & PHP" (plus petit, gris foncé)
- Bande ou carte principale : large rectangle à bords légèrement arrondis avec une couleur dominante rouge #dc2626, comme un bloc de hero section moderne
- Éléments secondaires : petits badges ou pastilles discrètes évoquant les technos (TypeScript, React, Node.js, Python, PHP) ou les services (Sites vitrines, SEO, IA & Data), en gris et rouge, minimalistes
- Éventuelle zone "CTA" stylisée (sans vrai bouton lisible) qui rappelle un bouton "Demander un devis", en rouge ou bordure rouge, pour suggérer l'action sans mettre de texte lisible
- Fond général : blanc ou gris très clair, avec de grands espaces vides pour rappeler une landing page propre

Style :
- Moderne, épuré, professionnel
- Identité proche du site danielcraft.fr : fond clair, accent rouge #dc2626, typographie type Inter
- Haute lisibilité pour le partage social
- Composition centrée ou légèrement décentrée, très aérée, avec un léger effet de profondeur via les ombres

Format : JPEG haute qualité, 1200x630px, ratio 1.91:1
```

## Prompt pour Canva (si tu préfères créer manuellement)

1. Template : "Facebook Post" ou "LinkedIn Post"
2. Dimensions : 1200x630px
3. Éléments à ajouter :
   - Fond : Blanc ou gris très clair, style page d'accueil propre
   - Bloc principal : grande carte ou bande horizontale rouge #dc2626 avec bords arrondis et légère ombre
   - Texte principal : "DanielCraft" (police Inter ou équivalent, gras, très foncé)
   - Texte secondaire : "Développeur Full-Stack TypeScript, Python & PHP" (gris foncé)
   - Petits badges ou icônes discrètes pour les technos (TypeScript, React, Node.js, Python, PHP) ou services (Sites vitrines, SEO, IA & Data)
   - Couleur accent principale : #dc2626 (rouge), complétée par des nuances de gris
4. Export : JPEG, qualité maximale

## Outils recommandés

- **Canva** : Template "Facebook Post", facile et rapide
- **Figma** : Plus de contrôle, gratuit
- **Midjourney** : Si tu veux quelque chose d'unique
- **DALL-E 3** : Via ChatGPT Plus, bon résultat
- **Stable Diffusion** : Si tu as une carte graphique

## Vérification

Une fois les images créées :
1. Vérifie les dimensions : 1200x630px exactement
2. Poids : < 1MB (optimise si nécessaire avec TinyPNG)
3. Noms et emplacements :
   - `assets/images/og/home-1200x630.jpg`
   - `assets/images/og/portfolio-1200x630.jpg`
   - `assets/images/og/prestations-1200x630.jpg`
   - `assets/images/og/metz-1200x630.jpg`
4. Test : Partage une URL sur Facebook pour voir le rendu

## Test de l'image

Une fois l'image en place, teste avec :
- [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/)
- [Twitter Card Validator](https://cards-dev.twitter.com/validator)
- [LinkedIn Post Inspector](https://www.linkedin.com/post-inspector/)
